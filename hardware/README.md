### IoTデバイスの役割
1. 入浴時間を監視・Webサーバ上に記録すること
2. 危険時の状況を検出すること

### IoTデバイスの機能
入浴時間をウェブ上に記録することができます。 \
距離センサやRFIDリーダを用いて、ユーザの行動を読み取り、ブザーと入力機器を用いてユーザとコミュニケーションをとることも可能です。
